Node.js 在调用某个package（包）时，会先检查包中的package.json文件的main字段，将其作为包的接口模块，如果package.json或者main字段不存在，会尝试寻找index.js或者index.node作为包的接口。

模 式 可通过 require 使用 注册PATH
本地模式 是 否
全局模式 否 是

# 全局->局部链接 ! not for windows !
# if npm install -g express
$ cd [project_folder]
$ npm link express # to use global package express as local package

# 局部->全局! not for windows !
# cd [package_folder]
$ npm link

# publish package
$ cd [package_folder]
$ npm init
$ npm publish
$ npm unpublish
